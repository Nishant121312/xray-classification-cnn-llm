Follow the steps to setup and run the project on your local computer.

- Create Environment
```
virtualenv .venv
```

- Active environment

on Linux
```
source .venv/Scripts/activate
```

- Install packages
```
pip install -r requirements.txt
```

- Start the app
```
streamlit run main/app.py
```

- Open the URL to view the app
```
http://localhost:8501/
```
